# Free-Tier Limits

The application relies strictly on free tiers:

- **Supabase:** Free plan (PostgreSQL, Auth, 500k Edge Function invocations).
- **Cloudflare Pages:** 100k requests/day.
- **Gemini AI:** Free-tier rate limits apply (configurable daily limits per user in the app).
- **Whisper:** Local processing costs nothing but utilizes user hardware.
